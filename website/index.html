NLM JATS @ Sourceforge

Welcome to the NLM JATS project on Sourceforge!

This is a space for exploring the NLM Journal Archiving and Interchange Tag Suite. It is a place for users and developers to collaborate and discuss tools and add-ons that might help to make JATS easier to use.

Note that this site is not officially connected to the NLM or the NISO standardization effort.
