%oasis.xmlns.attrib;

OASIS Table Model Namespace Declaration

Definition

Provides an attribute-like definition (name, declared value, and default value) for the namespace used with the xmlns:oasis attribute

Remarks

Namespace pseudo-attributes make the connection between a namespace prefix and a URL that represents the namespace. The %oasis.xmlns.attrib; Parameter Entity sets the value of this pseudo-attribute prefix to the URL for the OASIS (CALS) Table Exchange table model.

Declaration

<!ENTITY % oasis.xmlns.attrib
             "xmlns:oasis
                        CDATA      #FIXED
              '-//OASIS//DTD XML Exchange Table Model 19990315//EN'" >

Module

oasis-tablesetup.ent
