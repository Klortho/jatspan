Document Hierarchy Diagrams

This section contains diagrams of document structures defined by the OASIS Cals table model. The contents of the table cells are set by Parameter Entity and will vary depending on the tag set in which the <oasis:table> is used.

The diagrams are screen captures of the DTD(s) as displayed by the Near & Far® tool. The conventions of the Near & Far® format are described in the following section.
