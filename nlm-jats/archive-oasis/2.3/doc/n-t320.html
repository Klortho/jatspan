%oasis-tablesetup.ent;

OASIS XML Table Setup Module

Definition

External module named in the Archiving and Interchange Suite, but not used in any of the NLM Tag Sets, to set up the Parameter Entities needed by the OASIS (CALS) Table Exchange table model

Remarks

Authoring Note: The NLM Tag Sets use the XHTML table model, which is called by the %XHTMLtablesetup.ent; module which calls the %xhtml-table-1.mod; module to add the HTML 4.0 (XHTML) table tags to a tag set. If the OASIS (CALS) Table Exchange table model is wanted, the %oasis-tablesetup.ent; module should be invoked, which calls the %oasis-exchange.ent; module to add the OASIS table tags to a tag set.
