%tbl.table.att;

Table Attributes (OASIS Table Model)

Definition

Names additional attributes on the Table (<oasis:table>) element

Remarks

This Parameter Entity occurs in both the Table Setup and the OASIS Table Model modules. Since only the first Parameter Entity takes effect, the Table Setup module overrides the attribute listed in the Table Model module, and therefore, “pgwide” is not among the attributes below.

The official OASIS Exchange CALS Table Model documentation can be found at http://www.oasis-open.org/specs/a502.htm

<!ENTITY % tbl.table.att
             "%oasis.xmlns.attrib;
