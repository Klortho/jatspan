%tbl.table-titles.mdl;

Title of the Table (OASIS Table Model)

Definition

Sets the Title element inside a Table as an empty string, since the title is contained within the Table Wrapper (<table-wrap>) rather than within the OASIS CALS <oasis:table> element

Remarks

Note: This Parameter Entity has been set as an empty string to remove the <title> element from the <oasis:table> element’s content model.

The official OASIS Exchange CALS Table Model documentation can be found at http://www.oasis-open.org/specs/a502.htm
