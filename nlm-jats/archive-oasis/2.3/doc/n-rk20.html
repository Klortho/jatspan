xmlns:oasis

OASIS Namespace Declaration

Definition

This is not an attribute, but a namespace pseudo-attribute. The value associates a prefix for the OASIS Table Model elements with a URL that stands for the table model. The prefix “oasis” has been set to a URL for the OASIS (CALS) Table Exchange table model.

Attribute Values

In Element

<oasis:table> Table (OASIS Table Model)
Value Meaning
-//OASIS//DTD XML Exchange Table Model 19990315//EN The prefix “oasis” has been set to this formal public identifier (fpi), the fpi of the OASIS (CALS) Table Exchange table model.
Restriction: This attribute value may not be changed.
