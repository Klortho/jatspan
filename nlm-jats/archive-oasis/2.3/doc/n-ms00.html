<oasis:thead>

Table Header (OASIS Table Model)

Definition

The heading information for a group of columns (<oasis:tgroup>). A simple table that has only one <oasis:tgroup> will have only one <oasis:thead> which will serve as the header for the table. More complex tables may have a <oasis:thead> for each <oasis:tgroup>.

Remarks

A table may be divided into header elements (<oasis:thead>) and table body elements (<oasis:tbody>). Table headers may print or display in a different font or with a different alignment than the table body cells. Typically, the content of the <oasis:thead> appears at the top of the table group, and it may appear again at the top of any physical break in rows in the body, for example, at the top of each page of a multi-page <oasis:tgroup>.

The official OASIS Exchange CALS Table Model documentation can be found at http://www.oasis-open.org/specs/a502.htm

valign Vertical Alignment (OASIS Table Model)
