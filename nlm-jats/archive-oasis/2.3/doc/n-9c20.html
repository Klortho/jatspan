Definition

External module for the OASIS (CALS) Table Exchange table model, named in the Archiving and Interchange Suite, but not used in any of the NLM Tag Sets

Remarks

This is the standard OASIS (CALS) Table Exchange table model and is invoked from the %oasis-tablesetup.ent; module.
