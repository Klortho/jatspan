colnum

Column Number (OASIS Table Model)

Definition

The identifying number of a specific column in a table, i.e., a simple counting number

Remarks

The official OASIS Exchange CALS Table Model documentation can be found at http://www.oasis-open.org/specs/a502.htm

Attribute Values

In Element

<oasis:colspec> Column Specification (OASIS Table Model)
Value Meaning Behavior
A number The number of a specific column, for example, “1” or “4” The colnum identifies the specific column to the processing system.
Restriction: This attribute may be specified if the element is used.
