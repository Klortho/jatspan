cols

Number of Table Columns (OASIS Table Model)

Definition

Specifies the number of columns in a Table Group

Remarks

The official OASIS Exchange CALS Table Model documentation can be found at http://www.oasis-open.org/specs/a502.htm

Attribute Values

In Element

<oasis:tgroup> Table Group (OASIS Table Model)
Value Meaning
A number How many columns: a digit or series of digits corresponding to the desired number of columns
Restriction: This attribute must be specified if the element is used.
