%paracon;

Contents of an Entry (Table Cell) (OASIS Table Model)

Definition

Content model for a table cell (<oasis:entry>)

Remarks

This Parameter Entity is typically set to the NLM entity %inside-cell;, to control what will be in an <oasis:entry> element. Best Practice is to change the content of the %inside-cell; Parameter Entity to change the content of a cell and not to set the %paracon; Parameter Entity directly.

The official OASIS Exchange CALS Table Model documentation can be found at http://www.oasis-open.org/specs/a502.htm

Declaration

<!ENTITY % paracon      "#PCDATA | %inside-cell;"                    >
