%rendition-plus;

Rendition Markup Plus

Definition

In the base Suite, this Parameter Entity names a mix of the simplest of the typographic emphasis elements, as well as those for subscript and superscript, for use in models allowing data characters and this restricted mixture. In this Archiving Tag Set, this has been over-ridden to contain all the phrase-level elements.

Remarks

When this mix is enhanced in more permissive Tag Sets, best practice is for it to contain at least the typographic elements <sub> and <sup>.

DTD Maintenance Note: This Parameter Entity and the related Parameter Entity %just-rendition; have been put in place to restrict the amount of variability a person modifying the DTD through Parameter Entity redefinition can achieve. Some elements have been set as #PCDATA plus one Parameter Entity, while others have been set to #PCDATA plus the other, in an effort to allow designers to modify entire groups of elements, but not change similar models individually.

Declaration

<!ENTITY % rendition-plus
                        "| %all-phrase;"                             >
