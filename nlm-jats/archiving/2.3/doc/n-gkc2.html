%XHTMLtablesetup.ent;

XHTML Table Setup Module

Definition

External module in the Archiving and Interchange Suite that is named in the Module to Name the Modules (%modules.ent;) and then called from a DTD to set up the Parameter Entities needed by the XHTML table model

Remarks

Authoring Note: The Journal Archiving Tag Set uses the XHTML table model, which is called by the %XHTMLtablesetup.ent; module. This module then calls the %xhtml-table-1.mod; module to add the XHTML table tags to a tag set.

<!ENTITY % XHTMLtablesetup.ent
                        PUBLIC

"XHTMLtablesetup.ent"                                                >
