%list.class;

List Class Elements

Definition

The types of lists that may occur as part of a text, e.g., lists and definition lists

Remarks

This class excludes bibliographic reference lists (<ref-list>) because these have designated positions, such as within the back matter, and cannot occur anywhere a paragraph can occur.
