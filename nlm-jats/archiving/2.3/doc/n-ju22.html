%style.attrib;

Style Attribute (XHTML table model)

Definition

As part of an XHTML table, the attribute used to support inline style markup

Remarks

Since this Parameter Entity is defined in the public XHTML Tag Set, it does not follow the naming conventions for this Tag Set: the separator character is a period, not a hyphen.

Declaration

<!ENTITY % style.attrib
     "style        CDATA                    #IMPLIED"
>

Module

xhtml-inlstyle-1.mod
