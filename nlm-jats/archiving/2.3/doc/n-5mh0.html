alt-version

Alternate Version of a Graphic

Definition

Has an alternative version of a graphic been provided? For example, is there a thumbnail or a different resolution of the graphic?

Remarks

This attribute is only a flag and does not provide a location or an ID for an alternate version.

Attribute Values

In Element

<graphic> Graphic
Value Meaning
no There is no alternative version of this graphic.
yes There is an alternative version of this graphic.
Default value: no
