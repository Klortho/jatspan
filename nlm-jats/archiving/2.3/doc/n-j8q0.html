xmlns:mml

MathML Namespace Declaration

Definition

This is not an attribute, but the namespace pseudo-attribute. The value provides a prefix to use for the MathML elements. All namespace prefixes must be associated with a URL, and the prefix “mml” has been set to the URL of the World Wide Web Consortium (W3C) MathML 2.0 Tag Set.

Attribute Values

In Elements

<article> Article
<mml:math> Math (Mathml 2.0 Tag Set)
Value Meaning
http://www.w3.org/1998/Math/MathML The prefix “mml” has been set to this URL, the URL of the W3C MathML 2.0 Tag Set.
Restriction: This attribute value may not be changed.
