abbrev-type

Type of Journal Title Abbreviation

Definition

Holds the name of the authority (if known) that defined the abbreviated journal title, for example, “medline” for the MEDLINE abbreviation, “publisher” for a publisher-specific abbreviation, etc.

Attribute Values

In Element

<abbrev-journal-title> Abbreviated Journal Title
Value Meaning
Text, numbers, or special characters A name, acronym, or descriptive word naming the authority responsible for the abbreviation, for example, “publisher”, meaning that the publisher of the article also assigned the abbreviation
Restriction: This attribute may be specified if the element is used.
