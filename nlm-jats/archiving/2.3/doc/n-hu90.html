%date-model;

Date Elements Model

Definition

The content model for the <date> element

Remarks

The <string-date> element holds dates for which months and years are not given, for example, “apple blossom time”, “election day”, etc.

The <pub-date> elements holds dates an article was published or released in a particular format. While the Parameter Entity %pub-date-model; defaults to the standard%date-model; in the base Suite, %pub-date-model; has been over-ridden to have different content in this Archiving Tag Set.

Declaration

<!ENTITY % date-model   "( ( (day?, month?) | season)?,
                          year?, string-date?)"                      >
