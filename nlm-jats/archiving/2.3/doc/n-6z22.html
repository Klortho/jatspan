XHTML Table Inline Style Module

Definition

External module for the XHTML table model, supporting inline style markup; this module is named in the Module to Name the Modules (%modules.ent;) and then called from the XHMTL Table Setup Module (%XHTMLtablesetup.ent;) to include table elements in this Tag Set.

Declaration

<!ENTITY % xhtml-inlstyle-1.mod
                        PUBLIC
"-//W3C//ENTITIES XHTML Inline Style 1.0//EN"
"xhtml-inlstyle-1.mod"                                                  >
