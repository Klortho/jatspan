%rest-of-para.class;

Rest of Paragraph Class Elements (Minus Paragraph)

Definition

Information for the reader that is at the same structural level as a Paragraph (<p>). This Parameter Entity contains all paragraph-level objects except the paragraph element itself and those paragraph-level elements that are not used inside tables.

Declaration

<!ENTITY % rest-of-para.class
                        "ack | disp-quote | speech | statement |
                         verse-group | x"                            >

Module

archivecustom-classes.ent
