<supplement>

Supplement Information

Definition

This element is used in two ways:

  • For a journal published as a supplement, this is a wrapper element in the article metadata for all the information that is particular to that supplement, such as additional identification numbers, supplement titles, or supplement series information;
  • As part of a bibliographic reference (<citation> or <nlm-citation>), the wrapper element for all information particular to a supplement, for any work published as a supplement. Such information might include additional identification numbers, supplement titles, or supplement series information.

Remarks

For extensive examples of formatted <nlm-citation>s including use of <supplement>s in <nlm-citation>s, see: Sample PubMed Central Citations. To see tagged versions of these examples, see: Sample PubMed Central Citations - XML Tagged.

Attribute

supplement-type Type of Supplement

Model Information

Content Model

<!ELEMENT  supplement    (#PCDATA %supplement-elements;)*            >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • All the article linking elements:
    • Inline Supplementary Material <inline-supplementary-material>
    • Related Article Information <related-article>
  • Font <font>
  • Horizontal Rule <hr>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Overline Start <overline-start>
    • Overline End <overline-end>
    • Sans Serif <sans-serif>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
    • Underline Start <underline-start>
    • Underline End <underline-end>
  • Graphic, Inline <inline-graphic>
  • Private Character (Custom or Unicode) <private-char>
  • Formula, Inline <inline-formula>
  • Tex Math Equation <tex-math>
  • Math (Mathml 2.0 Tag Set) <mml:math>
  • Abbreviation or Acronym <abbrev>
  • Milestone End <milestone-end>
  • Milestone Start <milestone-start>
  • Named Special (Subject) Content <named-content>
  • All the simple link/cross-referencing elements:
    • Footnote <fn>
    • Target of an Internal Link <target>
    • X(cross) Reference <xref>
  • Subscript <sub>
  • Superscript <sup>
  • Contributor Group <contrib-group>
  • Title <title>

This element may be contained in:

<article-meta> Article Metadata; <citation> Citation; <nlm-citation> NLM Citation Model; <product> Product Information; <related-article> Related Article Information
