%label.class;

Label Class Elements

Definition

Class comprising the <label> element, which can be used to hold the pre-generated or author-provided number or symbol of a labeled object such as a footnote (“3” or “*”) or a table wrapper (“Table 6”)

Remarks

The Label Class Elements Parameter Entity was created as a means of constraining use of the <label> element in the Publishing and Authoring DTDs.

Declaration

<!ENTITY % label.class  "label"                                      >
