currency

Currency

Definition

In what currency is this price presented?

Remarks

Best Practice: Although this is an optional attribute, best practice would be to include it for every price where it is known. The attribute can take any value, but best practice is to use the ISO 4117 “currency code” values. These are 3-letter, all upper case codes that are NOT the same as the ISO country codes. Examples include:

USD

United States Dollar in the US and US territories

CAD

Canadian Dollar

EUR

European Union Euro

RUB

Russian Ruble
These codes are defined by ISO in ISO 4217 and listed at: http://www.iso.org/iso/en/prods-services/popstds/currencycodeslist.html#SZ.

Attribute Values

In Element

<price> Price
Value Meaning
Text, numbers, or special characters A standardized abbreviation for a currency (for best practice, the ISO 4117 “currency code” values)
Restriction: This attribute may be specified if the element is used.
