%inline-supplementary-material-atts;

Inline Supplementary Material Attributes

Definition

Attribute list for the <inline-supplementary-material> element, which is an inline reference to supplementary material such as a dataset or video

Remarks

This element was created to accommodate a small number of Tag Sets and is expected to be very rarely used.
