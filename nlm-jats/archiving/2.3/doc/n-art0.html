<page-range>

Page Ranges

Definition

A container element for additional page information to record discontinuous pages ranges, such as “8-11, 14-19, 40” (i.e., an article begins on page 8, runs through 11, skips to pages 14 through 19, and concludes on page 40.

Remarks

The <page-range> element only supplements other page elements and DOES NOT replace <fpage> and <lpage>. The <fpage> element and the <lpage> element (where available) should always be tagged; infrastructures for linking references across publishers (such as that of CrossRef) use first and last page information on an article as part of their identification process. Accordingly, material with a page range should be tagged: 

         <fpage>8</fpage>
         <lpage>40</lpage>
         <page-range>8-11, 14-19, 40</page-range>

Attribute

content-type Type of Content

Related Elements

A number of elements in the Suite relate to page numbers:

  • <fpage> names the page number on which the article begins;
  • <lpage> names the page number on which the article ends (which should be the same page number or a number larger than the starting page number);
  • <elocation-id> replaces the start and end page elements just described for electronic-only journals; and
  • <page-range> records discontinuous page ranges (See note below.).
  • <page-count> holds the total page count, if the publisher has provided one. Typically this element records what the publisher said and makes no validity claim.
