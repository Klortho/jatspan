supplement-type

Type of Supplement

Definition

Indicates the type of or reason for a supplement (if the publisher has recorded that information), for example:

issue

for supplements to a particular journal issue

conference

for supplements containing information from a conference, symposium, or other gathering

periodic

for supplements issued at regular intervals

Attribute Values

In Element

<supplement> Supplement Information
Value Meaning
Text, numbers, or special characters The type or reason for the supplement, for example, “conference”
Restriction: This attribute may be specified if the element is used.
