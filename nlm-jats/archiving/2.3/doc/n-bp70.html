<publisher>

Publisher

Definition

Wrapper element for information about the person, company, or other entity which published a work

Attribute

content-type Type of Content

Model Information

Content Model

<!ELEMENT  publisher    %publisher-model;                            >

Description

The following, in order:

  • Publisher’s Name <publisher-name>
  • Publisher’s Location <publisher-loc>, zero or one

This element may be contained in:

<journal-meta> Journal Metadata
