%simple-intable-display.class;

Simple Table Display Class Elements

Definition

Simple display items allowed, for example, inside a table wrapper. This is a very similar construction to the %simple-display.class; Parameter Entity, but with the <array> element excluded since an array is not permitted within a table wrapper <table-wrap>.

Declaration

<!ENTITY % simple-intable-display.class
                        "chem-struct | graphic | media | preformat"  >

Module

default-classes.ent
