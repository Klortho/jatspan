<journal-meta>

Journal Metadata

Definition

Metadata that identifies the journal in which the article was published

Related Elements

Older, MAJOUR-style journal DTDs split the metadata for a journal article into three categories: journal metadata, article metadata, and issue metadata. In this Journal Archiving Tag Set, the metadata specific to the journal is within the element Journal Metadata <journal-meta>, and the metadata specific both to the article and to the particular issue of the journal (if any) is within the Article Metadata <article-meta> element.

Model Information

Content Model

<!ELEMENT  journal-meta %journal-meta-model;                         >

Description

The following, in order:

  • Journal Identifier <journal-id>, zero or more
  • Journal Title (Full) <journal-title>, zero or more
  • Journal Subtitle (Full) <journal-subtitle>, zero or more
  • Translated Title <trans-title>, zero or more
  • Translated Subtitle <trans-subtitle>, zero or more
  • Abbreviated Journal Title <abbrev-journal-title>, zero or more
  • ISSN <issn>, zero or more
  • Publisher <publisher>, zero or one
  • Notes <notes>, zero or one
  • Custom Metadata Wrapper <custom-meta-wrap>, zero or one

This element may be contained in:

<front> Front Matter
