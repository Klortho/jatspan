%address.class;

Address Class Elements

Definition

All of the elements found only inside an address

Remarks

An address could also include address linking elements that could be found in both an address and other elements, for example, <email>, <ext-link>, and <uri>.

Declaration

<!ENTITY % address.class
                        "addr-line | country | fax |
                         institution | phone "                       >

Module

default-classes.ent
