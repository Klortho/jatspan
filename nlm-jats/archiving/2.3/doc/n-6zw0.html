<front-stub>

Stub Front Metadata

Definition

A reduced metadata set for use in sub-articles and responses, which will inherit metadata not defined in the stub from the enclosing article

Model Information

Content Model

<!ELEMENT  front-stub   %front-stub-model;                           >

Description

The following, in order:

  • Article Identifier <article-id>, zero or more
  • Title Group <title-group>, zero or one
  • Any combination of:
    • Contributor Group <contrib-group>
    • Affiliation <aff>
    • X - Generated Text and Punctuation <x>
  • Author Note Group <author-notes>, zero or one
  • Optionally any one of:
    • The following, in order:
      • First Page <fpage>
      • Last Page <lpage>, zero or one
      • Page Ranges <page-range>, zero or one
    • Electronic Location Identifier <elocation-id>
  • History: Document History <history>, zero or one
  • Permissions <permissions>, zero or one
  • Abstract <abstract>, zero or more

This element may be contained in:

<response> Response; <sub-article> Sub-article
