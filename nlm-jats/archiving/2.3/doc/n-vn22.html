%table.class;

Table Class Elements

Definition

Elements that will be used to contain the rows and columns inside the <table-wrap> element. The following elements have been set up for inclusion: the XHTML table model <table> element.

Remarks

In the original draft of this DTD, the OASIS Exchange Table model was also included and some NLM Tag Set supersets choose to use this model instead of or in addition to the XHTML table model.

Declaration

<!ENTITY % table.class  "table"                                      >
