<app-group>

Appendix Group

Definition

Wrapper element for one or more appendices (<app> elements), which hold additional or ancillary material attached to an article

Remarks

As set up in the Journal Archiving Tag Set, the Appendix Group element can contain both appendices and interspersed bibliographic reference lists, because some older journal DTDs make use of this model.

This element is the same as the Appendix Matter <appm> element used in AAP- and ISO 12080-derivative DTDs.

Attributes

content-type Type of Content
id Identifier

Model Information

Content Model

<!ELEMENT  app-group    %app-group-model;                            >

Description

The following, in order:

  • Title <title>, zero or one
  • Any combination of:
    • Array (Simple Tabular Array) <array>
    • Boxed Text <boxed-text>
    • Chemical Structure (Display) <chem-struct>
    • Chemical Structure Wrapper <chem-struct-wrapper>
    • Figure <fig>
    • Figure Group <fig-group>
    • Graphic <graphic>
    • Media Object <media>
    • Preformatted Text <preformat>
    • Supplementary Material <supplementary-material>
    • Table Wrapper <table-wrap>
    • Table Wrapper Group <table-wrap-group>
    • Formula, Display <disp-formula>
    • Paragraph <p>
    • Definition List <def-list>
    • List <list>
    • Tex Math Equation <tex-math>
    • Math (Mathml 2.0 Tag Set) <mml:math>
    • Related Article Information <related-article>
    • Acknowledgments <ack>
    • Quote, Displayed <disp-quote>
    • Speech <speech>
    • Statement, Formal <statement>
    • Verse Form for Poetry <verse-group>
    • X - Generated Text and Punctuation <x>
  • Any combination of:
    • Appendix <app>
    • Reference List (Bibliographic) <ref-list>
