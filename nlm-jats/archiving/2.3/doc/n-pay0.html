dtd-version

Version of the DTD

Definition

Identifies which version of the tag set, named in the DOCTYPE declaration, was used to make the article under consideration, which therefore should be used to validate this article

Remarks

Authoring Note: This is a “fixed” attribute (meaning that the value is set via a Parameter Entity in the DTD; thus, an author or editor cannot change it) that should change every time the tag set changes versions or revisions.

Attribute Values

In Element

<article> Article
Value Meaning
2.3 The 2.3 version of the Journal Archiving Tag Set is being used.
Restriction: This attribute value may not be changed.
