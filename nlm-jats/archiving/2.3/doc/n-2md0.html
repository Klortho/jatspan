<tr>

Table Row (XHTML table model)

Definition

See W3C documentation for the XHTML table model.

A container element for all the cells in a single table row

Attributes

align Horizontal Alignment (XHTML table model)
alternate-form-of Alternate Form of Graphic, Media Object, Etc.
char Table Cell Alignment Character (XHTML table model)
charoff Table Character Offset (XHTML table model)
content-type Type of Content
id Identifier
style Style (XHTML table model; MathML 2.0 Tag Set)
valign Vertical Alignment (XHTML table model)

Model Information

Content Model

<!ELEMENT %tr.qname;  %tr.content; >

Description

One or more of any of:

  • Table Header Cell (XHTML table model) <th>
  • Table Data Cell (XHTML table model) <td>

This element may be contained in:

<table> Table (XHTML table model); <tbody> Table Body (XHTML table model); <tfoot> Table Footer (XHTML table model); <thead> Table Header (XHTML table model)
