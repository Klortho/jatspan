Default Element Mixes Module

Definition

External module in the Archiving and Interchange Suite that sets up the default Parameter Entities and element mix definitions used in content models for the Archiving and Interchange Suite

Remarks

It is expected that some or all of the Parameter Entities declared in this module will be overridden in the DTD-specific custom mixes module, %archivecustom-mixes.ent;.
