%note.class;

Note Class Elements

Definition

Class consisting solely of the <note> element

Remarks

A note may appear at the end of an Article or at the end of a Table. For example, a typical end-of-article note is a “Note in Proof”.

Declaration

<!ENTITY % note.class   "note"                                       >
