alt-title-type

Type of Alternate Title

Definition

Names the reason for or purpose of an alternate title, for example, “short”, meaning a short version of the title, “toc” for use in a Table of Contents, “ASCII” for an ASCII title, “running” for a right-running-head title for a print publication, etc.

Attribute Values

In Element

<alt-title> Alternate Title
Value Meaning
Text, numbers, or special characters A word or phrase describing the rationale for this version of the article title, for example, “right-running-head”
Restriction: This attribute may be specified if the element is used.
