%block-display.class;

Block Display Class Elements

Definition

Names graphical or other image-related elements in the Archiving and Interchange Suite that cannot be used inline, but only as block elements; these display elements may occur within the text of a table cell or paragraph although they are typically at the same hierarchical level as a paragraph.

Declaration

<!ENTITY % block-display.class
                        "array | boxed-text | chem-struct |
                         chem-struct-wrapper | fig | fig-group |
                         graphic | media | preformat |
                         supplementary-material | table-wrap |
                         table-wrap-group"                           >

Module
