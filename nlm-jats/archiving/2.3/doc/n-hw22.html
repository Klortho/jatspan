%sub-article-model;

Subarticle Model

Definition

Content model for the <sub-article> element

Remarks

The <floats-wrap> element used in this model is a wrapper to hold the floating objects (figures, tables, text boxes, graphics, etc.) that occur anywhere in the subarticle, where these floating objects are collected together rather than interspersed throughout the subarticle where they are referenced. This element was added to accommodate publishers who like to group all such floating objects together.

Declaration

<!ENTITY % sub-article-model
                        "( (front | front-stub), body?, back?,
                           floats-wrap?, (sub-article* | response*) )"
                                                                     >

Module
