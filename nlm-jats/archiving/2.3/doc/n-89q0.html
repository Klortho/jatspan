DTD-Specific Models/Attributes Customizations Module

Definition

External module in the Journal Archiving DTD (Green) that sets up the Parameter Entities for DTD-specific over-rides of element groups and complete content models, as well as attribute list and value over-rides

Remarks

These parameter entities will over-ride selected content models and attribute lists from the base Suite for the Journal Archiving DTD (Green).
