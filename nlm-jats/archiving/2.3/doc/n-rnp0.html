units

Units

Definition

Expresses the unit of measure in which the size (length) of a cited work is expressed, for example, pages, minutes, seconds, linear feet, etc.

Attribute Values

In Element

<size> Size
Value Meaning
Text, numbers, or special characters Names the units of measure in which the size is expressed
Restriction: This attribute must be specified if the element is used.
