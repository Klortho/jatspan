%simple-phrase;

Simple Phrase-Level Textual Elements

Definition

In the base Suite, this Parameter Entity names elements that may be used almost anywhere text is used, for example, inside most kinds of titles. In this Archiving Tag Set, this Parameter Entity has been over-ridden to allow any phrase-level elements.

Element

Remarks

In the base Suite, this Parameter Entity is slightly more complex and contains more elements than the %simple-text; Parameter Entity; this Parameter Entity would be used for potentially complex models.

Declaration

<!ENTITY % simple-phrase
                        "| %all-phrase;"                             >
