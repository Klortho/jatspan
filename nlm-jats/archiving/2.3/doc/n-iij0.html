pub-type

Type of Publication

Definition

Used to record the type of publication that was released on a particular date, for example, a print-only publication versus an electronic-only publication

Attribute Values

In Elements

<issn> ISSN
<pub-date> Publication Date
Value Meaning
Text, numbers, or special characters The type of print or electronic publication, for example, “epub” for electronic publication. This attribute may take any value, but see the list of suggested values below.
Restriction: This attribute may be specified if the element is used.

Suggested usage

Although designed to accept any text as its value, potential values for this attribute might include:

epub

Electronic publication (usually web, but also includes CD-ROM or other electronic only distribution)

ppub

Print publication

epub-ppub

Both print and electronic publications

epreprint

Electronic preprint dissemination

ppreprint

Print preprint dissemination

ecorrected

Corrected in electronic

pcorrected

Corrected in print

eretracted

Retracted in electronic

pretracted

Retracted in print
