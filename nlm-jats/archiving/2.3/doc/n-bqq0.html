%appearance.class;

Appearance Class Elements

Definition

Rarely used elements that govern stylistic or typographic considerations

Remarks

These elements were added by the inclusion of the XHTML table model. Since this Archiving Tag Set seeks to preserve intellectual content, not physical appearance, best practice is to avoid using these elements.

Declaration

<!ENTITY % appearance.class
                        "font | hr"                                  >
