%mathmlsetup.ent;

MathML Setup Module

Definition

External module in the Archiving and Interchange Suite that is named in the Module to Name the Modules (%modules.ent;) and then called from the Archiving Tag Set to set up the namespaces and Parameter Entities for the MathML elements

Remarks

This module then calls the MathML modules to include the MathML elements in the tag set.

Declaration

<!ENTITY % mathmlsetup.ent
                        PUBLIC
"-//NLM//DTD Archiving and Interchange Suite MathML Setup Module v2.3 20070202//EN"
"mathmlsetup.ent"                                                    >
