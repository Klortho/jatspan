<sig-block>

Signature Block

Definition

An area of text and graphic material placed at the end of the body of an article to hold the graphical signature or description of the person(s) responsible for or attesting to the content

Remarks

This element was deliberately left out of the original NLM DTDs because it was thought (and analysis seemed to back this up) that a <sig-block> element would merely duplicate information in the metadata and could be generated. Further samples, especially in books, indicate that signature blocks are not as closely tied to contributor metadata as was supposed, and further, that a structure to hold the material would be very useful for already published works.

Attributes

content-type Type of Content
id Identifier

Model Information

Content Model

<!ELEMENT  sig-block    (#PCDATA %sig-block-elements;)*              >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • All the article linking elements:
    • Inline Supplementary Material <inline-supplementary-material>
    • Related Article Information <related-article>
  • Font <font>
  • Horizontal Rule <hr>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Overline Start <overline-start>
    • Overline End <overline-end>
    • Sans Serif <sans-serif>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
    • Underline Start <underline-start>
    • Underline End <underline-end>
  • Graphic, Inline <inline-graphic>
  • Private Character (Custom or Unicode) <private-char>
  • Formula, Inline <inline-formula>
  • Tex Math Equation <tex-math>
  • Math (Mathml 2.0 Tag Set) <mml:math>
  • Abbreviation or Acronym <abbrev>
  • Milestone End <milestone-end>
  • Milestone Start <milestone-start>
  • Named Special (Subject) Content <named-content>
  • All the simple link/cross-referencing elements:
    • Footnote <fn>
    • Target of an Internal Link <target>
    • X(cross) Reference <xref>
  • Subscript <sub>
  • Superscript <sup>
  • Line Break <break>
  • Graphic <graphic>
  • Media Object <media>
  • Signature <sig>

This element may be contained in:

<body> Body of the Article
