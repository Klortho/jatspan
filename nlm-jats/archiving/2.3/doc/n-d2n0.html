<string-name>

Name of Person (Unstructured)

Definition

Wrapper element for personal names where the stricter organization of the <name> element cannot be followed.

This is a very loose element, which may contain text, numbers, special characters, generated text, and any or all of the naming elements, such as <surname>.

Remarks

Conversion Note: Use this element if some name parts are unknown or untagged, for example, <string-name>Prince Charles</string-name>. Using the <string-name> element is better practice than merely leaving a person’s name untagged.

The Name of Person (Unstructured) model permits the tagging of name parts within it, so a <surname> could be tagged within a Name of Person (Unstructured). Examples of when name part tagging inside <string-name> would be appropriate include:

  • A scenario in which “string names” are tagged as part of a preliminary process, and subsequent processes include tagging the specific name parts; or
  • An instance in which reverse order of a name, with the separating comma, is to be preserved as a single sting, but name parts still need to be tagged for enhanced search capability, i.e., “Abernathy, the Honorable Sir Edward” might be tagged as 
    <string-name><surname>Abernathy</surname>, the Honorable Sir Edward</string-name>

Model Information

Content Model

<!ELEMENT  string-name  (#PCDATA %string-name-elements;)*            >

Description

Any combination of:

  • Text, numbers, or special characters
  • All the address linking elements:
    • Email Address <email>
    • External Link <ext-link>
    • Uniform Resource Indicator (URI) <uri>
  • All the article linking elements:
    • Inline Supplementary Material <inline-supplementary-material>
    • Related Article Information <related-article>
  • Font <font>
  • Horizontal Rule <hr>
  • All the emphasis elements:
    • Bold <bold>
    • Italic <italic>
    • Monospace Text (Typewriter Text) <monospace>
    • Overline <overline>
    • Overline Start <overline-start>
    • Overline End <overline-end>
    • Sans Serif <sans-serif>
    • Small Caps <sc>
    • Strike Through <strike>
    • Underline <underline>
    • Underline Start <underline-start>
    • Underline End <underline-end>
  • Graphic, Inline <inline-graphic>
  • Private Character (Custom or Unicode) <private-char>
  • Formula, Inline <inline-formula>
  • Tex Math Equation <tex-math>
  • Math (Mathml 2.0 Tag Set) <mml:math>
  • Abbreviation or Acronym <abbrev>
  • Milestone End <milestone-end>
  • Milestone Start <milestone-start>
  • Named Special (Subject) Content <named-content>
  • All the simple link/cross-referencing elements:
    • Footnote <fn>
    • Target of an Internal Link <target>
    • X(cross) Reference <xref>
  • Subscript <sub>
  • Superscript <sup>
  • Degree(s) <degrees>
  • Given (First) Names <given-names>
  • Prefix <prefix>
  • Surname <surname>
  • Suffix <suffix>
  • X - Generated Text and Punctuation <x>

This element may be contained in:

<citation> Citation; <contrib> Contributor; <person-group> Person Group for a Cited Publication; <product> Product Information; <related-article> Related Article Information
