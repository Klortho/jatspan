arrange

Arrangement of Subscripts and Superscripts

Definition

Indicates whether multiple subscripts or superscripts applying to the same character should stack, i.e., be placed one above the other as compactly as possible, or stagger, i.e., be stretched out in a line

Attribute Values

In Elements

<sub> Subscript
<sup> Superscript
Value Meaning
stack The multiple subscripts or superscripts should be placed one above the other as compactly as possible.
stagger The multiple subscripts or superscripts should be stretched out in a line.
Restriction: This attribute may be specified if the element is used.
