<def-item>

Definition List: Definition Item

Definition

One entry in a two-part list. The entry holds a term and one or more of the definitions/descriptions of that term.

Remarks

Authoring Note: While this element contains an optional Label (of a Figure, Reference, Etc.) element, the Label (of a Figure, Reference, Etc.) element should be included only in those circumstances where a formatting override is needed; Label (of a Figure, Reference, Etc.) should NOT be used in the ordinary course of tagging.

Attribute

id Identifier

Related Elements

A Definition List may have a Title naming the entire list, followed by optional headings for terms and their definitions. <term-head> contains the heading over the first (term) column, and <def-head> contains the heading for the second (definition) column; neither serves as a container element for terms or definitions. Individual <term>s and their <def>s are paired in <def-item>s.

Model Information

Content Model

<!ELEMENT  def-item     %def-item-model;                             >

Description

The following, in order:

  • Label (of a Figure, Reference, Etc.) <label>, zero or one
  • Definition List: Term <term>, zero or more
  • Any combination of:
    • Definition List: Definition <def>
    • X - Generated Text and Punctuation <x>

This element may be contained in:

<def-list> Definition List
