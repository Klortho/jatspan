<gloss-group>

Glossary Group

Definition

A distinct grouping of glossary definitions, usually preceded by a title that identifies the grouping (e.g., “Definitions for Medical Students”, “Definitions for Nurses”)

The intent of glossary groups is to allow glossaries to be divided into headed sections, recursively deep, where each section can be about a special topic or category of terms.

Remarks

Typical content of such a Glossary Group is a title that names the group, a paragraph of explanation that defines the group or explains any exclusions, and then a series of term/definition pairs tagged inside a Definition List <def-list>.

Attributes

content-type Type of Content
id Identifier

Model Information

Content Model

Description

The following, in order:

  • Title <title>, zero or one
  • Any combination of:
    • Array (Simple Tabular Array) <array>
    • Boxed Text <boxed-text>
    • Chemical Structure (Display) <chem-struct>
    • Chemical Structure Wrapper <chem-struct-wrapper>
    • Figure <fig>
    • Figure Group <fig-group>
    • Graphic <graphic>
    • Media Object <media>
    • Preformatted Text <preformat>
    • Supplementary Material <supplementary-material>
    • Table Wrapper <table-wrap>
    • Table Wrapper Group <table-wrap-group>
    • Formula, Display <disp-formula>
    • Paragraph <p>
    • Definition List <def-list>
    • List <list>
    • Tex Math Equation <tex-math>
    • Math (Mathml 2.0 Tag Set) <mml:math>
    • Related Article Information <related-article>
    • Acknowledgments <ack>
    • Quote, Displayed <disp-quote>
    • Speech <speech>
    • Statement, Formal <statement>
    • Verse Form for Poetry <verse-group>
    • X - Generated Text and Punctuation <x>
