pub-id-type

Type of Publication Identifier

Definition

Names the type of identifier, or the organization or system that defined the identifier, for the identifier of a journal article, other work, or component element

Usage

This attribute is used in three quite different contexts: 1) as part of the article metadata (for example, on the <article-id> element, which holds an identifier for the entire article), 2) inside a bibliographic citation (for example, on the <pub-id> element, which identifies a publication cited in a bibliographic reference list), and 3) on an object identifier (for example, on the <object-id> element, which can act as an identifier for any number of different elements).

Attribute Values

In Elements

<article-id> Article Identifier; <issue-id> Issue Identifier; <object-id> Object Identifier; <pub-id> Publication Identifier for a Cited Publication; <volume-id> Volume Identifier
Value Meaning
Text, numbers, or special characters The type of identifier “doi” or name or type of organization that created the identifier “aggregator”. This attribute may take any value, but see the list of suggested values below.
Restriction: This attribute may be specified if the element is used.

Suggested usage

Values for this attribute might include:

archive

Identifier assigned by an archive or other repository (generally used with elements <object-id>, <issue-id>, and <volume-id>)

art-access-id

Generic article accession identifier for interchange and retrieval between archives

aggregator

Identifier assigned by a data aggregator (generally used with elements <object-id>, <issue-id>, and <volume-id>)

coden

Obsolete PDB/CCDC identifier (may be present on older articles)

doaj

Directory of Open Access Journals

doi

Digital Object Identifier

index

Identifier assigned by an abstracting or indexing service (generally used with elements <object-id>, <issue-id>, and <volume-id>)

manuscript

Identifier assigned to a manuscript

medline

NLM MEDLINE identifier

pii

Publisher Item Identifier; see http://pubs.acs.org/epub/piius.htm or http://www.aip.org/epub/piipr.html

pmc (or pmcid)

PubMed Central identifier (generally used with elements <object-id>, <issue-id>, and <volume-id>)

pmid

PubMed ID; see www.ncbi.nlm.nih.gov/entrez/ query.fcgi?db=PubMed

publisher-id

Publisher’s identifier, such as an “article-id”, “artnum”, “identifier”, “article-number”, “pub-id”, etc.

sici

Serial Item and Contribution Identifier (A journal article may have more than one SICI, e.g., one for a print version and another for an electronic version.)
