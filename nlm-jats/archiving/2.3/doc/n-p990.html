%floats-display.class;

Floating Display Class Elements

Definition

Block display elements that can take the “position” attribute, and thus may be floating as opposed to fixed in narrative position in an <article>, <sub-article>, or <response>

Remarks

This class was created for use as the content of the <floats-wrap> element.

Declaration

<!ENTITY % floats-display.class
                        "boxed-text |  chem-struct-wrapper |
                         fig | fig-group | graphic | media |
                         preformat | supplementary-material |
                         table-wrap | table-wrap-group"              >

Module

default-classes.ent
