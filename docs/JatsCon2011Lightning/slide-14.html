The JATS Bundle is ...

... JATS!

A new distribution, but the same DTDs. Each DTD is still composed of the exact same set of modules. All the public identifiers defined in the exist distributions will resolve correctly. The data models should be exactly the same.

Disclaimer: Every effort has been made to make sure that no changes were made that would effect the ability to use these in the same way that the distributions from the NLM are used. These have been thoroughly tested to verify that they are the same, but I cannot guarantee that there are no differences.

Changes included:

  • Fixing relative system identifiers so that modules that refer to each other can still be used without a catalog file,
  • Fixing discrepencies and problems in the existing NLM distributions. For example, system identifiers that used the wrong case (so would not work on *nix systems.)

Detailed list of all the substantive changes will be posted soon on jatspan.org.

If you find any mistakes / problems, please tell me!
